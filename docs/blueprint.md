# **App Name**: CryptoExist

## Core Features:

- News Feed: Display a paginated list of cryptocurrency news articles with titles, excerpts, and categories.
- Article Details: Show the full content of a selected news article, including the title, author, publication date, and category.
- Market Ticker: Display a live ticker with cryptocurrency prices and 24h changes.
- Crypto Converter: Real-time conversion tool for converting Indonesian Rupiah (IDR) to Bitcoin (BTC).
- Fear & Greed Index: Visualize the Crypto Fear & Greed Index using a gauge.
- News Fetching API: API endpoint to fetch paginated news articles from the PostgreSQL database, including category and author data.
- Market Ticker API: API endpoint to fetch cryptocurrency market ticker data.

## Style Guidelines:

- Background color: Deep Black (#0A0A0A) for the main background.
- Card background color: Dark Grey (#121212) for news cards and other content containers.
- Primary color: Electric Yellow (#FFD700) for headlines, buttons, and the market ticker.
- Accent color: Electric Cyan (#00F0FF) for hover states, graph lines, and technical accents.
- Text color: White (#FFFFFF) for primary text and Slate Grey (#94a3b8) for secondary text.
- Headline font: 'Space Grotesk' sans-serif for headlines and larger text elements, to give a techy and scientific feel. Body font: 'Inter' sans-serif for body copy.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use 'Lucide React' icons throughout the application for a consistent and modern look.
- Implement a Bento-grid layout for the HeroSection to showcase a main article and trending news.
- Use subtle hover animations on NewsCard components to highlight user interaction with Electric Cyan border glow.
